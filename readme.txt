Create multiple galleries, each with it's own page.

== Description ==

With this plugin you can create multiple galleries, all filed under "Gallery" page. 
Very easy to use, just install, and start adding galleries in your Gallery section in admin. 
Pictures are added in each gallery with the help of WordPress's Media Uploader. If you want a certain image to show up as the gallery cover instead a random one, set it as featured image.
You can select two sizes of the images: one size for gallery cover images, another for thumbnails shown in each gallery page.

== Installation ==

1. Upload the contents of `AWGallery` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure the default settings are good for you. (WP Admin -> Gallery -> Settings)

== Frequently Asked Questions ==

= How do I add a new gallery =

Each post created in Gallery tab in WP Admin is regarded as a gallery. All of it's images, uploaded with WordPress's Media Uploader, are regarded as the content of the gallery.
